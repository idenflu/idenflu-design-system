Icons

작업 의미를 보조하는 SVG symbol과 icon-only 접근성 기준입니다.

Component standard

Icons 문서의 공통 구조.

Anatomy

기본 구성요소를 정리해 구현과 디자인 리뷰의 출발점을 맞춥니다.

  • 24px viewBox
  • symbol id
  • stroke rule
  • usage label
  • accessible wrapper
Variants

Action, status, object, settings, permission symbols

States

Default, hover, focus, selected, loading, disabled, error 또는 locked 상태를 같은 density로 비교합니다.

Accessibility

Keyboard path, visible focus, semantic role, accessible name, helper/error 연결을 함께 검토합니다.

Examples

단독 컴포넌트보다 idenflu 제품 화면 안에서 쓰이는 예시를 우선합니다.

QA

Visual QA에서 light/dark, compact density, disabled hover, focus, long text overflow를 다시 확인합니다.

Document template

Icons 문서는 의미, 사용 위치, accessible wrapper를 같은 순서로 봅니다.

01 Overview

아이콘이 action, object, status 중 어떤 의미인지 먼저 정합니다.

02 Anatomy

24px viewBox, symbol id, stroke, wrapper, accessible name을 분리합니다.

03 Variants

Action, status, object, permission icon을 용도별로 묶습니다.

04 States

Icon-only button 안에서는 hover, focus, disabled 상태를 버튼과 함께 봅니다.

05 Accessibility

장식 아이콘은 aria-hidden, 단독 액션은 aria-label을 사용합니다.

06 QA

깨진 symbol, 잘못된 stroke, 없는 accessible name을 확인합니다.

Review checklist
  • 아이콘만으로 상태를 설명하지 않고 텍스트 label을 함께 둡니다.
  • Icon-only button은 동사와 대상을 포함한 aria-label을 사용합니다.
  • 새 symbol은 tokens.json과 icons.svg에 동시에 등록합니다.

Token contract

아이콘은 symbol registry와 stroke rule을 토큰으로 관리합니다.

DecisionTokenRule
ViewBox{icons.viewBox}모든 symbol은 0 0 24 24 기준으로 정렬합니다.
Stroke{icons.strokeWidth}1.8px stroke, square cap, miter join을 유지합니다.
ColorcurrentColor아이콘 색은 부모 버튼이나 상태 text color를 따릅니다.
Registry{icons.symbols}문서에 쓰는 symbol id는 token registry와 일치해야 합니다.

API contract

Icon API 구현 계약.

TypeContract

Statesdecorative, informative, action, disabled, selected

Token usage

Icon tokens를 실제 token path에 연결합니다.

{icons.viewBox} {icons.strokeWidth} {icons.strokeLinecap} {icons.strokeLinejoin}

Usage guidance

idenflu 화면에서 쓰는 기준을 먼저 정합니다.

Use

작업 의미를 빠르게 스캔하게 하거나 텍스트 action을 보조할 때 씁니다.

Avoid

상태, 위험도, 권한을 색상 아이콘 하나만으로 설명하지 않습니다.

Replace with

의미 전달이 중요하면 visible text, badge, feedback copy를 함께 둡니다.
