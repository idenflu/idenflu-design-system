Menu

Contextual actions, sorting, filtering, and danger commands.

Component standard

Menu 문서의 공통 구조.

Anatomy

Trigger, surface, group label, item, checked state, danger item을 나눕니다.

  • Trigger
  • menu surface
  • item group
  • shortcut
  • danger action
Variants

Action menu, sort menu, filter menu, row menu, destructive menu

States

Closed, open, hover, focus, checked, disabled, danger 상태를 같은 밀도로 비교합니다.

Accessibility

aria-haspopup, aria-expanded, role menu, Escape close, focus restore를 확인합니다.

Examples

테이블 row action, 정렬, 필터, export 같은 반복 작업 안에서 검토합니다.

QA

긴 item label, disabled item, danger action, keyboard order, mobile overflow를 확인합니다.

Campaign row menu
Review checklist
  • 주요 CTA는 메뉴 안에 숨기지 않습니다.
  • 위험 액션은 group 끝에 두고 색과 문구를 함께 사용합니다.
  • 메뉴를 닫으면 trigger로 focus가 돌아갑니다.

Token contract

메뉴는 작은 표면, hairline, compact typography로 구성합니다.

DecisionTokenRule
Surface{colors.surface-raised}메뉴는 떠 있는 표면이지만 과한 그림자보다 hairline으로 구분합니다.
Item height{sizing.control-height}각 item은 버튼 높이와 같은 기준으로 tab target을 유지합니다.
Text{typography.body-sm}보조 액션은 13px body-sm를 사용해 테이블 밀도와 맞춥니다.
Danger{colors.accent-coral}위험 액션은 coral과 명시적인 action label을 함께 사용합니다.
