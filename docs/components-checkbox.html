Checkbox

여러 항목을 동시에 켜고 끄는 다중 선택과 boolean 선택 기준입니다.

Component standard

Checkbox 문서의 공통 구조.

Anatomy

Box, checkmark, label, description, helper text를 나눕니다.

  • box
  • checkmark
  • label
  • description
  • helper text
Variants

Single checkbox, checkbox group, select-all, with description, small size를 구분합니다.

States

Unchecked, checked, indeterminate, focus, disabled, error 상태를 비교합니다.

Accessibility

native checkbox input, visible label, aria-checked=mixed, space 토글을 지킵니다.

Examples

채널 필터, 권한 선택, 동의 checkbox에 사용합니다.

QA

긴 label, indeterminate 대비, disabled, dark contrast, focus ring을 확인합니다.

Context example

Checkbox는 여러 항목을 동시에 선택할 수 있게 보여줍니다.

Review checklist
  • 여러 항목을 동시에 선택할 수 있을 때 checkbox를 사용합니다.
  • 정확히 하나만 선택해야 하면 radio group을 사용합니다.
  • 부분 선택 상태는 aria-checked=mixed로 전달합니다.

Token contract

Checkbox는 box radius와 primary accent를 사용합니다.

DecisionTokenRule
Box{radius.sm}Box는 작은 radius로 control과 구분합니다.
Checked{colors.primary}Checked와 indeterminate는 primary로 채웁니다.
Border{colors.hairline}Unchecked box는 hairline border만 씁니다.

API contract

Checkbox API 구현 계약.

TypeContract
Propslabel, checked, description, helperText, indeterminate, size, disabled
Statesunchecked, checked, indeterminate, focus, disabled, error

Usage guidance

idenflu 화면에서 쓰는 기준을 먼저 정합니다.

Use

여러 항목을 동시에 켜고 끄거나 단일 boolean 동의처럼 현재 선택 값이 화면에 계속 남아야 할 때 씁니다.

Avoid

정확히 하나만 선택해야 하는 경우나 즉시 실행 명령을 checkbox로 표현하지 않습니다.

Replace with

단일 선택은 radio group, 즉시 적용 설정은 switch, 모드 전환은 segmented control 또는 tabs, 일회성 명령은 button을 씁니다.

Checkbox

여러 항목을 동시에 켜고 끄는 다중 선택을 표현합니다.

Checkbox group

React: <Checkbox label="Instagram reels" /> — 네이티브 <input type="checkbox">를 custom box와 checkmark로 스타일링하며 label, description, helperText, indeterminate, size props를 지원합니다.

부분 선택 상위 항목은 indeterminate로 두고 aria-checked="mixed"를 전달합니다. 단일 boolean 동의에는 <Checkbox label="이용약관에 동의합니다" />처럼 한 개만 사용합니다.
