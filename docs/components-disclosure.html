Token contract

Disclosure는 button, border, surface token을 함께 씁니다.

DecisionTokenRule
Trigger{sizing.control-height}Trigger hit area는 control 높이를 유지합니다.
Panel{colors.surface-raised}Panel은 trigger 아래 같은 surface 계층에 둡니다.
Divider{colors.hairline}Accordion 항목은 hairline으로 구분합니다.
Focus{colors.primary}Trigger focus는 panel 상태와 별개로 유지합니다.

Usage guidance

idenflu 화면에서 쓰는 기준을 먼저 정합니다.

Use

상세 사유, audit, helper, row expansion처럼 맥락 안에서 정보를 접고 펼칠 때 씁니다.

Avoid

주요 action이나 필수 오류를 접힌 영역 안에 숨기지 않습니다.

Replace with

결정 표면은 overlay, 화면 전환은 navigation, 동일 화면 모드는 tabs를 씁니다.
