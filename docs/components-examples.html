Usage guidance

idenflu 화면에서 쓰는 기준을 먼저 정합니다.

Use

컴포넌트 조합이 실제 idenflu 운영 흐름에서 어떻게 작동하는지 검증할 때 씁니다.

Avoid

예쁜 단품 샘플만 나열하거나 실패/권한/빈 상태를 생략하지 않습니다.

Replace with

개별 규칙은 각 component page, 전체 화면 판단은 patterns 또는 visual QA로 분리합니다.
