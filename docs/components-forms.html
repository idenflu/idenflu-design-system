Token contract

Forms는 field와 surface, feedback token을 함께 씁니다.

DecisionTokenRule
Field{sizing.field-height}입력 control은 field 높이를 유지합니다.
Group{spacing.lg}Field group 간격은 section 안에서 일관되게 둡니다.
Surface{colors.surface-raised}Save bar와 validation summary는 surface-raised 위에 둡니다.
Error{colors.accent-coral}오류는 summary와 field helper에서 같은 tone으로 연결합니다.

Usage guidance

idenflu 화면에서 쓰는 기준을 먼저 정합니다.

Use

캠페인 설정, 승인 사유, 권한 변경처럼 여러 field가 하나의 저장 action으로 묶일 때 씁니다.

Avoid

단일 inline 수정이나 읽기 전용 metadata를 form으로 과장하지 않습니다.

Replace with

단일 값은 input/select, 빠른 on/off는 controls, 복잡한 결정은 overlay form을 씁니다.
