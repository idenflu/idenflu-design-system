Switch

즉시 적용되는 on/off 설정을 표현하는 toggle 기준입니다.

Component standard

Switch 문서의 공통 구조.

Anatomy

Track, thumb, label, description, helper text를 나눕니다.

  • track
  • thumb
  • label
  • description
  • helper text
Variants

Default switch, with description, small size를 구분합니다.

States

Off, on, focus, disabled, error 상태를 같은 density로 비교합니다.

Accessibility

role=switch, visible label, aria-checked, space 토글을 지킵니다.

Examples

설정 패널, 리뷰 필터, 알림 토글 같은 제품 화면 안에서 보여줍니다.

QA

긴 label, disabled, dark contrast, focus ring을 확인합니다.

Context example

Switch는 즉시 반영되는 설정을 명확히 보여줍니다.

Review checklist
  • Switch는 즉시 적용되는 설정에만 사용합니다.
  • 저장 버튼이 필요한 값에는 사용하지 않습니다.
  • 현재 상태가 화면에 계속 남아 있어야 할 때만 씁니다.

Token contract

Switch는 control height와 primary accent를 사용합니다.

DecisionTokenRule
Size{sizing.control-height}Track 높이는 control-height 기준으로 맞춥니다.
On{colors.primary}켜진 상태는 primary로 채웁니다.
Off{colors.surface-2}꺼진 상태는 surface-2 track을 씁니다.

API contract

Switch API 구현 계약.

TypeContract
Propslabel, checked, description, helperText, size, disabled
Statesoff, on, focus, disabled, error
A11ynative checkbox role switch, visible label, aria-checked, keyboard space toggle

Token usage

Switch tokens를 실제 token path에 연결합니다.

{sizing.control-height} {colors.primary} {colors.surface-2} {colors.hairline} {motion.duration.base}

Usage guidance

idenflu 화면에서 쓰는 기준을 먼저 정합니다.

Use

즉시 적용되는 on/off 설정처럼 현재 값이 화면에 계속 남아야 할 때 씁니다.

Avoid

저장 버튼을 눌러야 반영되는 설정이나 즉시 실행 명령을 switch로 표현하지 않습니다.

Replace with

다중 선택은 checkbox, 모드 전환은 segmented control 또는 tabs, 일회성 명령은 button을 씁니다.

Switch

즉시 적용되는 on/off 설정을 토글합니다.

Switches

React: <Switch label="Auto-assign reviewers" /> — 네이티브 <input type="checkbox" role="switch">를 pill로 스타일링합니다. label, description, helperText, size props와 controlled/uncontrolled를 지원합니다.
