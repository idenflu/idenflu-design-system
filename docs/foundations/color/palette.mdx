## Code

> CSS 변수를 제공하지만, 제품 코드에서는 `Color` 토큰 대신 `Themes`의 시멘틱 토큰 사용을 권장합니다.

```css
@import "@idenflu/ui-tokens/css/variables.css";

.example {
  color: var(--base-gray-900);
  background: var(--brand-500);
  border: 1px solid var(--border-subtle);
}
```

```ts
// JS object — optional for non-CSS tooling
import tokens from "@idenflu/ui-tokens";
```
