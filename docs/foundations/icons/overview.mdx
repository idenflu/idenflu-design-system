# Icons

Icons는 UI에서 동작을 암시하고 상태를 보조하는 **시각 기호**입니다.  
Idenflu 아이콘은 `@idenflu/ui-icons` 스프라이트와 `Icon` 컴포넌트로 제공되며, 색은 `currentColor`로 상속합니다.

<br />
<br />

## 원칙

1. **스프라이트 단일 소스**  
   모든 심볼은 `icons.svg`의 `<symbol>`로 관리하고, `<use href="…#name">`로 참조합니다.  
   인라인 path를 컴포넌트마다 복제하지 않습니다.

2. **색은 토큰으로**  
   아이콘에 전용 color prop을 두지 않습니다. 부모의 `color`(권장: Theme text 토큰)를 상속합니다.

3. **접근성 역할 분리**  
   장식 아이콘은 `aria-hidden`이 기본입니다. 단독으로 의미를 전달할 때만 `label`을 줍니다.  
   `IconButton` / `Button` 안에서는 버튼이 accessible name을 담당합니다.

## 사용 가이드

### Do

- `name`은 `IconName` union만 사용합니다.
- 앱 루트에서 `IconSpriteProvider`로 번들러가 해석한 sprite URL을 한 번 주입합니다.
- 색은 `var(--text-*)` 등 Theme 토큰으로 맞춥니다.

### Don't

- 제품 코드에 raw SVG path를 복제하지 않습니다.
- 버튼 안의 아이콘에 중복 `label`을 주지 않습니다(버튼 이름을 씁니다).
- hex로 아이콘 색을 하드코딩하지 않습니다.

## Code

```tsx
import spriteUrl from "@idenflu/ui-icons/icons.svg?url";
import { Icon, IconSpriteProvider } from "@idenflu/ui-react";

function App() {
  return (
    <IconSpriteProvider href={spriteUrl}>
      <Icon name="search" size="medium" />
      <Icon
        name="warning"
        label="경고"
        style={{ color: "var(--text-warning)" }}
      />
    </IconSpriteProvider>
  );
}
```

```ts
import { iconNames, getIconHref, type IconName } from "@idenflu/ui-icons";

const href = getIconHref("search"); // "@idenflu/ui-icons/icons.svg#search"
```
