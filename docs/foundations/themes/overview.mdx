# Themes

Themes는 Primitive Color를 화면 **역할**에 연결하는 시맨틱 토큰입니다.  
컴포넌트는 hex나 `Base.*`를 직접 고르지 않고, `Surface` · `Background` · `Border` · `Text`로 라이트/다크와 상태를 일관되게 표현합니다.

<br />
<br />

## 원칙

1. **역할로 색을 고른다**  
   버튼 배경은 `Brand.500`이 아니라 `background-primary`입니다. 의미가 바뀌면 토큰 이름을 바꿉니다.

2. **테마 전환에 안전한 구조**  
   컴포넌트는 동일한 토큰 이름을 쓰고, light/dark JSON이 참조 값만 바꿉니다.

3. **상태는 같은 계열로 확장**  
   `error` → `error-hover` → `error-active`, `subtle`, `tint`처럼 의미 축을 유지합니다.

<br />
<br />

## 토큰 계층

```text
Primitive (Base / Brand / Transparent)
    ↓
Themes (Background / Surface / Border / Text)
    ↓
Components (Button, Alert, Input, …)
```

## 미리보기

Light/Dark와 역할 탭으로 대표 토큰을 확인합니다. 값은 CSS 변수가 테마에 따라 해석됩니다.

## 사용 가이드

### Do

- 제품 코드에서는 Primitive보다 Themes 토큰을 우선 사용합니다.
- 같은 의미의 상태는 같은 계열로 확장합니다.
- 텍스트 대비는 실제 배경 토큰과 함께 검증합니다.

### Don't

- 컴포넌트 CSS에 HEX·RGB·Primitive를 직접 쓰지 않습니다.
- 라이트/다크별 스타일을 컴포넌트 안에 중복 정의하지 않습니다.
- 상태 토큰 없이 hover/active 색을 따로 만들지 않습니다.

## Code

```css
@import "@idenflu/ui-tokens/css/variables.css";

.panel {
  background: var(--surface-00);
  color: var(--text-primary);
  border: 1px solid var(--border-subtle);
}

.action {
  background: var(--background-primary);
  color: var(--text-on-color);
}

.action:hover {
  background: var(--background-primary-hover);
}
```

`variables.css`는 `:root`(Light)와 `.dark`(Dark)에 **같은 시맨틱 토큰 이름**을 선언합니다. 컴포넌트는 `var(--surface-00)`만 쓰면 되고, 테마는 클래스 범위로 바뀝니다.

```css
/* @idenflu/ui-tokens/css/variables.css (구조) */
:root {
  /* light.json → semantic aliases */
  --surface-00: var(--base-white);
  --text-primary: var(--base-gray-900);
  --border-subtle: var(--base-gray-200);
}

.dark {
  /* dark.json → 동일 토큰 이름을 Dark 참조로 재정의 */
  --surface-00: …;
  --text-primary: …;
  --border-subtle: …;
}
```

`.dark`는 `html` / `body` 또는 하위 컨테이너에 올리면 됩니다.

```html
<!-- 문서 전체 -->
<html class="dark">
  <body>
    <div class="panel">…</div>
  </body>
</html>
```

```tsx
// 문서 루트 토글
document.documentElement.classList.toggle("dark", isDark);

// 하위 트리만 토글
<div className={isDark ? "dark" : undefined}>
  <Panel />
</div>;
```
