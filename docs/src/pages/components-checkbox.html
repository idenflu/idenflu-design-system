Context example

Checkbox는 여러 항목을 동시에 선택할 수 있게 보여줍니다.

Review checklist
  • 여러 항목을 동시에 선택할 수 있을 때 checkbox를 사용합니다.
  • 정확히 하나만 선택해야 하면 radio group을 사용합니다.

Checkbox

여러 항목을 동시에 켜고 끄는 다중 선택을 표현합니다.

Checkbox group

React: <Checkbox label="Instagram reels" /> — 네이티브 <input type="checkbox">를 custom box와 checkmark로 스타일링하며 label, description, helperText, indeterminate, size props를 지원합니다.

부분 선택 상위 항목은 indeterminate로 두고 aria-checked="mixed"를 전달합니다. 단일 boolean 동의에는 <Checkbox label="이용약관에 동의합니다" />처럼 한 개만 사용합니다.
