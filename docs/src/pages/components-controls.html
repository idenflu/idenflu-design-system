Review checklist
  • 선택값이 하나면 segmented control, 여러 개면 checkbox group을 사용합니다.
  • Switch는 즉시 적용되는 설정에만 사용합니다.

Choose a control

어떤 control을 쓸지 한눈에 비교합니다.

각 control은 전용 페이지에서 anatomy, variants, states, token contract까지 자세히 다룹니다. 아래 비교표로 먼저 맞는 control을 고르고 상세 문서로 이동하세요. Pick the right control here, then open its dedicated page for the full spec.

Switch

즉시 적용되는 on/off 설정 하나를 토글할 때. Use for a single setting that takes effect immediately — no save step.

React: <Switch label="Auto-assign reviewers" />

Switch 상세 문서로 이동 →

Checkbox

목록에서 0개 이상을 다중 선택하거나 단일 동의를 받을 때. Use to select zero or more items, or a single opt-in (supports indeterminate).

React: <Checkbox label="Instagram reels" />

Checkbox 상세 문서로 이동 →

Radio group

여러 옵션 중 정확히 하나만, 그리고 옵션이 모두 보여야 할 때. Use when exactly one option must be chosen from a fully visible set.

React: <RadioGroup label="Approval routing" options={[...]} value onChange />

Radio group 상세 문서로 이동 →

Segmented control

2~4개의 상호 배타적인 뷰를 한 줄에서 즉시 전환할 때. Use for 2–4 mutually exclusive views switched inline, all visible at once.

React: <SegmentedControl label="Time range" options={[...]} value onChange />

Segmented control 상세 문서로 이동 →

Range

연속적인 수치 범위에서 한 값을 선택할 때. 현재 값을 항상 숫자로 함께 보여 줍니다. Use to pick one value across a continuous numeric range; always echo the value as text.
