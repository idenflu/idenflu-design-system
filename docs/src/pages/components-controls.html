Controls

필터, 토글, 선택 상태를 밀도 있게 제어합니다.

Segmented control

대시보드 기간, 채널, 리뷰 상태 전환에 사용합니다.

React: <SegmentedControl label="Time range" options={[...]} value onChange /> — 단일 선택, role="group" + aria-pressed.

Switches

React: <Switch label="Auto-assign reviewers" /> — 네이티브 <input type="checkbox" role="switch">를 pill로 스타일링합니다. label, description, helperText, size props와 controlled/uncontrolled를 지원합니다.

Checkbox group

React: <Checkbox label="Instagram reels" />(indeterminate 지원)와 <RadioGroup options={[...]} />(fieldset/legend group label).
