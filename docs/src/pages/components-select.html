선택지는 짧게 유지하고 결과 상태를 바로 노출합니다.

React: <Select label="Review stage" options={[...]} placeholder="Select stage" error helperText required /> — 네이티브 <select>를 label·helper·error와 연결하고 SelectOption.disabled로 비활성 옵션을 표시합니다. expanded·multiple·searchable 또는 SelectOption.icon/SelectOption.description를 주면 native control 대신 커스텀 role="listbox" 폼으로 전환합니다. 아래 데모의 요약 패널은 문서용 인터랙션이며, 컴포넌트는 정적 helperText/error만 노출합니다.

Expanded select

아이콘·설명·다중 선택·검색이 필요하면 expanded listbox 폼으로 전환합니다.

Native <select>는 짧고 단순한 단일 선택에 쓰고, 옵션에 아이콘·보조 설명이 필요하거나 다중 선택·검색이 필요하면 role="listbox" 팝업으로 바꿉니다. 아래 데모는 React <Select>의 expanded 계약을 문서용 vanilla JS로 재현합니다.
