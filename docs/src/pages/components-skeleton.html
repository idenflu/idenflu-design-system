Review checklist
  • Skeleton 블록은 실제 콘텐츠와 같은 자리와 개수를 차지합니다.
  • 데이터가 준비되면 즉시 실제 콘텐츠로 교체하고 placeholder를 남기지 않습니다.
  • 장식 블록은 aria-hidden으로 숨기고 옆에 visible Loading 문구를 둡니다.
  • reduced motion 환경에서는 shimmer를 멈추고 정적 면으로 보여 줍니다.
