Open
  1. Trigger stores restore target.
  2. aria-expanded becomes true.
  3. Overlay becomes visible.
  4. First safe focus target receives focus.
Navigate
  1. Tab order stays inside modal or drawer.
  2. Esc closes when it is non-destructive.
  3. Menu arrow keys move active menuitem.
  4. Validation sends focus to the first invalid field.
Close
  1. Overlay hidden state is restored.
  2. aria-expanded becomes false.
  3. Focus returns to trigger.
  4. Selection/filter context remains unchanged.
